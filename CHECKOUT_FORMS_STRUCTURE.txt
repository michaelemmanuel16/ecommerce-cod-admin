COMPONENT FILES
===============
1. /frontend/src/pages/CheckoutForms.tsx
   - Main page component
   - Forms list/grid
   - CRUD operations orchestration

2. /frontend/src/components/forms/CheckoutFormBuilder.tsx
   - Modal container
   - Form builder with all sections
   - react-hook-form integration
   - State management for fields/packages/upsells

3. /frontend/src/components/forms/FormFieldEditor.tsx
   - Single field configuration row
   - 12-column grid layout
   - Field type handling

4. /frontend/src/components/forms/PackageEditor.tsx
   - Single package configuration row
   - 12-column grid layout

5. /frontend/src/components/forms/UpsellEditor.tsx
   - Single upsell configuration row
   - 12-column grid layout

TYPE DEFINITIONS
================
/frontend/src/types/checkout-form.ts

- FieldType: 'text' | 'phone' | 'textarea' | 'select'
- FormField: { id, label, type, required, enabled, options? }
- ProductPackage: { id, name, price, quantity, popular }
- Upsell: { id, name, price, quantity, popular }
- CheckoutFormSettings: { defaultCountry }
- CheckoutFormStyling: { buttonColor, accentColor }
- CheckoutForm: Complete form entity

COLOR SCHEME
============
Button Color (default): #0f172a (slate-900)
Accent Color (default): #f97316 (orange-500)
Active Badge: Green (bg-green-100, text-green-800)
Inactive Badge: Gray (bg-gray-100, text-gray-800)
