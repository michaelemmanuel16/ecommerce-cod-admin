## How to Capture Missing Screenshots

2. Navigate to http://localhost:5173
3. Log in as rep1@codadmin.com / password123
4. Navigate to each page/modal and capture screenshots
5. Save with the naming convention above
6. Optionally annotate screenshots with arrows/highlights for clarity

## Notes

- Screenshots should be taken at standard desktop resolution (1280x720 or similar)
- Use clear, readable UI states (no loading spinners if possible)
- Ensure sensitive data is test data only
- Consider adding annotations (arrows, boxes, highlights) to draw attention to key features
